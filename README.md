# Panther‑Claw AI — Warm & Wise Guide

Panther‑Claw AI — это тёплый и мудрый проводник, созданный на базе Gemini 3. Он объединяет стратегию, психологическую поддержку, символизм, творчество и аналитику в одном интеллектуальном агенте.

## Возможности
- Стратегия и принятие решений
- Психологическая поддержка
- Ритуалы и символы
- Творческие тексты и легенды
- Анализ ситуаций и данных
- Мягкий, глубокий, образный стиль общения

## Технологии
- Gemini 3 API
- AI Studio Agent
- Contextual Reasoning
- Generative Text

## Архитектура
User → AI Studio Agent → Gemini 3 Model  
            ↳ Reasoning  
            ↳ Generation  
            ↳ Context  
            ↳ Tools

## Запуск
1. Открыть Google AI Studio
2. Создать нового агента
3. Вставить содержимое файла system-prompt.txt в System Instructions
4. Включить публичный доступ
5. Сохранить проект

## Explanation
- AI Studio Agent управляет поведением ассистента.
- Gemini 3 выполняет рассуждение, генерацию и анализ.
- System Prompt задаёт характер Panther‑Claw AI.

- # Example Gemini 3 API call (pseudo-code)

from gemini import GeminiClient

client = GeminiClient(api_key="YOUR_KEY")

response = client.generate(
    model="gemini-3",
    prompt="Создай ритуал для начала нового этапа."
)

print(response.text)
